
* 这里描述一个纯手工开发的一个简单的hello world项目
```bash
# 创建一个maven项目
$ mvn archetype:generate -DgroupId=imooc-arthur -DartifactId=spring-mvc-study -DarchetypeArtifactId=maven-archetype-webapp  -DarchetypeCatalog=internal

$ cd spring-mvc-study

# 并将目录结构改为本文档所在目录下的 spring-mvc-study.tar.gz 的目录结构

$ mvn jetty:run

# 访问地址 localhost:8080/hello/mvc
```
