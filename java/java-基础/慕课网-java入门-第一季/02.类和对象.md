
* 所有 Java 程序都以类(class)为组织单元

* 构造方法
    * 与类名相同，且没有返回值
    ```
    public 构造方法名(){}
    ```

* 类成员变量初始化
```    
public class HelloWorld{
    private int a;
    private int b;
    static private int c;

    // 通过 {} 来初始化普通成员变量
    {
        a = 10;
        b = 10;
    }
    // 通过 static {} 来初始化静态成员变量
    static {
        c = 10;
    }
}
```
