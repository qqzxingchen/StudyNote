
* XML 可扩展的标记语言，它是 SGML 的一个子集
    * 当前使用的是 XMl 1.0 版本
    * XML与HTML的区别
        * XML 中的元素是可扩展，但是 HTML 中的元素是固定的
        * HTML 可以用浏览器来解析执行，XML 的解析器需要自己来做

* XML 作用
    * 程序的配置文件
    * 数据交换：不同语言之间交换数据
    * 小型数据库

* XML 文档的组成部分
    * XML 文档声明
        ```xml
        <?xml version="1.0" encoding="utf-8" standalone="no" ?>
        <!--
            文档声明必须位于文档第一行
            version xml版本（当前一般都是 1.0）
            encoding 编码格式
            standalone 是否存在约束
                如果 standalone 为 no，则说明该文档没有约束，可以随便定义元素结构
                如果 standalone 为 yes，则说明该文档有约束，元素结构是有一些约束存在的
            -->
        ```
    * XML 处理指令
    * XML 元素
        * XML 文档只能存在一个根元素
    * XML 特殊字符和 CDATA 区
    * XML 注释

* XML 约束
    * 通过问xml文档头的 standalone 来为 xml 文档添加约束
    * 约束分为两种
        * DTD 约束
            * DTD 是平面式文档，dtd文件不是xml文档，通常文件后缀名为 .dtd ，是最早的 xml 约束
        * Schema 约束
            * Schema 本身也是 xml 文档，它比 dtd 文档更强大，通常扩展名为 .xsd，它是新的xml文档约束，用来替代 dtd

* DTD
    * DTD的引入
        * 在 XML 文档中直接定义 DTD
        * 编写 .dtd 文件
        * 从外界直接引入 .dtd 文件
    * xml 中直接定义 DTD
    ```DTD
    <!DOCTYPE students[
        <!ELEMENT students(student+)>
        <!ELEMENT student(name,age,gender)>

    ]>


    ```
